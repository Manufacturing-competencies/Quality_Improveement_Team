7 Alat Kualitas Dasar

  1. Checksheet

    Digunakan untuk mencatat dan mengumpulkan data secara sistematis.

  2. Histogram

    Diagram batang untuk melihat distribusi data dan pola variasi.

  3. Control Chart

    Membantu memantau kestabilan proses dari waktu ke waktu.

  4. Scatter Diagram

    Menunjukkan hubungan antara dua variabel dan kecenderungan data.

  5. Pareto Diagram

    Memprioritaskan masalah utama berdasarkan prinsip 80/20.

  6. Ishikawa Diagram

    Diagram sebab-akibat untuk mengidentifikasi akar masalah.

  7. Flowchart

    Visualisasi langkah-langkah proses secara berurutan.
